## Como usar (via maven)?

<pre>
&lt;dependency&gt;
  &lt;groupId&gt;com.github.kyriosdata.bsus&lt;/groupId&gt;
  &lt;artifactId&gt;bsus&lt;/artifactId&gt;
  &lt;version&gt;1.0.0&lt;/version&gt;
&lt;/dependency&gt;
</pre>
